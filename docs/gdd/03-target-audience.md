# 03 — Público-Alvo

## Personas

### Família (pagante principal B2C)

- Quer conteúdo seguro, sem anúncios e sem compras escondidas.
- Valoriza progresso mensurável (“meu filho está melhorando”).
- Prefere assinatura simples ou compra de módulo à loja cheia de IAPs.

### Criança / adolescente (jogador)

- Sessões curtas; feedback imediato.
- Interface grande, touch-friendly, pouca leitura onde possível (faixas menores).

### Adulto autodidata

- Quer desafio real (capitais, países, biomas).
- Usa modos Contra o Tempo / Sobrevivência / Desafio Diário.

### Professor / escola (B2B)

- Precisa de visão de turma, atividades e relatórios.
- Ranking por evolução, não por “quem ganha dos outros”.

### Terapeuta / psicopedagogo (futuro)

- Atividades calibradas por faixa etária e objetivo cognitivo.
- Sessões controladas, sem pressão.

## Adequação por faixa

| Faixa | Foco | Geografia / mapas |
|---|---|---|
| 0–1 | Estimulação visual/auditiva, descoberta | Não |
| 1–2 | Formas, cores, animais, grande/pequeno… | Não |
| 2–4 | Memória, associação, contagem, puzzles | Limitado / futuro |
| 4–6 | Letras, números, lógica | Introdução com apoio |
| 6–10 | Geo, história, ciências, idiomas, matemática | Sim |
| 10+ / adultos | Desafios completos | Sim + expansões |
