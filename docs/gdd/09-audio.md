# 09 — Áudio

## Princípios

- Sons suaves, curtos, não repetitivos de forma irritante.
- Volume padrão baixo; mute global fácil.
- Áudio nunca é obrigatório para jogar.

## Eventos (Geografia)

| Evento | Áudio |
|---|---|
| Acerto | Tom curto ascendente, suave |
| Erro | Tom neutro / descendente leve — sem “buzzer” agressivo |
| Fim de sessão | Sting curto positivo |
| UI (tap) | Opcional, muito discreto |

## Música

- Opcional; loops calmos só em menus se necessário.
- Desligada por padrão em perfis de bebê / foco.

## Futuro

- TTS para nomes de regiões (acessibilidade + faixas 4–6).
- Pacotes de áudio por módulo sem acoplar à engine.
